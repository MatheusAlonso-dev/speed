# Speed

A simple browser-based running game developed with HTML, CSS, and JavaScript, focused on animations and basic game mechanics.

## Purpose

This project was created for learning purposes, with a focus on practicing animations, DOM manipulation, and game logic using pure front-end technologies.

## Features

- Character movement
- Obstacle generation
- Collision detection
- Score system
- Continuous running environment

## Technologies

- HTML
- CSS (animations)
- JavaScript

## Requirements

- A modern web browser

## How to Run

1. Open the project folder
2. Run the `index.html` file in your browser
